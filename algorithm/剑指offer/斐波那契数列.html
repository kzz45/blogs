# 斐波那契数列

  • LeetCode地址open in new window
斐波那契数 （通常用 F(n) 表示）形成的序列称为 斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
给定 n ，请计算 F(n) 。

# 递归

递归的话，会有重复计算的问题 O = 2^n

class Solution:
    def fib(self, n):
        if n < 2:
            return n
        return self.fib(n - 1) + self.fib(n - 2)

# 动态规划

避免了重复计算 O = n

class Solution:
    def fib(self, n):
        if n < 2:
            return n
        p, q = 0, 1
        ret = 0
        for i in range(0, n - 1):
            ret = p + q
            p = q
            q = ret
        return ret
