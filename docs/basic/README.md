## [链表唯手熟尔](./02.md)

【链表 Linked List】是一种线性数据结构，插入删除高效，查询访问效率低下

分单链表、双链表、循环链表

双链表就是比单链表多了一个 prev 指针，循环链表就是就是首尾相连

![](./asset/linked_list.png)

初始化，或者叫链表的结点定义

```py
# 链表结点类
class ListNode:
    def __init__(self, x):
        self.val = x # 结点值
        self.next = None # 指向下一结点的指针（引用）
# 双向链表结点类
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None # 指向后继结点的指针（引用）
        self.prev = None  # 指向前驱结点的指针（引用
```
