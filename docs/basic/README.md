## [骚气的位运算](01.md)

记录位运算的起因是最近开始看算法的内容，忽然发现其实在平时工作中也用到了，但是没有仔细的了解。

所以在使用中就会有一种装逼的嫌疑，并且这玩意儿确实高效。
