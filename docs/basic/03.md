# 链表

![](./asset/linked_list.png)

- 单链表

```py
class ListNode:
    def __init__(self, val=0):
        self.val = val
        self.next = None
```

- 双链表

```py
class ListNode:
    def __init__(self, val=0):
        self.val = val
        self.next = None
        self.prev = None
```
