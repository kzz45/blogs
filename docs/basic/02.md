# 数组

![](./asset/array.png)

::: tip
数组使用一组连续的内存空间，来存储一组具有相同类型的数据。

数组的最大特点的支持随机访问

访问元素、改变元素的时间复杂度为 O1

在尾部插入、删除元素的时间复杂度也是 O1

普通情况下插入、删除元素的时间复杂度为 On
:::

## 重点

- 数组：二分查找
- 数组：移除元素
- 数组：有序数组的平方
- 数组：长度最小的子数组
- 数组：螺旋矩阵 II

## 数组初始化

```py
foo = [0]*5
bar = [0 for _ in range(100)]
```

## 数组遍历

```py
# 通过索引遍历数组
for i in range(len(arr)):
    print(arr[i])
```

## 数组查找

```py
# 在数组中查找指定元素
def find(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
```
